01 — Overview
What this course is about
This course introduces contemporary methods in machine learning for social scientific inquiry. We address the technological arms race between information generation—AI-created content like deepfakes, podcasts, and synthetic media—and the analytical tools needed to study it. Each class combines lecture with a hands-on workshop, culminating in final presentations and a paper.
01

Regression & Linear Algebra

OLS, logistic regression, LASSO/Ridge, NumPy fundamentals, gradient descent optimization, and the mathematical foundations for everything that follows.

02

Tree-Based Methods

Decision trees, random forests, gradient-boosting (XGBoost, LightGBM, CatBoost), and model interpretation with SHAP values.

03

Deep Learning

Neural network fundamentals in PyTorch, word embeddings, and transformer architectures from BERT to GPT.

04

LLM Augmentation

Zero-shot classification, local models via Ollama, and GPT workflows for labeling, topic modeling, and synthetic data generation.

05

Agentic Workflows

Autonomous AI agents with Claude Code: skills systems, tool integration, and building end-to-end pipelines that act on the world.

06

Networks & RL

Social network analysis with NetworkX, centrality measures, graph visualization, and reinforcement learning fundamentals.

Submission requirements
Each component is graded against an explicit rubric. The four pieces below make up the 35% Final Deliverables weight.
PAPER

PNAS-style paper (8 pages, LaTeX)

Build in Overleaf from the PNAS template gallery — select the single-column mathematics article template (second from left), then click Open as Template to copy it into your account. Follow the PNAS author guidelines for figures, references, and formatting. Remove the DRAFT watermark by editing the .cls file (set display watermark = false) — a small deduction applies if it stays. Ignore the template’s “feel free to exceed 10 pages” note: the limit is 8 pages. Sections: Introduction & related work, Data, Methods, Results (writing + figures/tables), Discussion, and the Agentic Analysis section — AI assistant transcripts plus a critical reflection on what you asked, accepted, rejected, and where the AI failed. Submit a compiled PDF alongside the .tex source.

REPO

Public GitHub repository

Numbered sequential notebooks (00_pull.ipynb, 01_merge.ipynb, …). README documenting inputs / function / outputs per notebook. Directory layout: code/, data/ (or cloud link), output/. No spaces in filenames, no hardcoded paths, functions defined at notebook start, diagnostic output before and after every data merge.

WEB

Public project website

Narrative arc: question → data → method → result → takeaway. Live and publicly accessible by the deadline; embed key figures. Platform is not flexible — build the site in React and deploy via Vercel. Pushes after the deadline = zero for the website component.

TALK

6-minute presentation

Live in class Wed 08/26 (last day). Hard 6-minute stop. Slides or live website demo. Followed by one instructor question with a 30-second response window. Focus on the headline finding and the evidence that supports it.

05 — Policies
Course expectations

Academic Honor

Violations result in a failing grade. Proper attribution is required per Dartmouth's Institute for Writing and Rhetoric standards.

AI Tools Policy

You may use ChatGPT and other AI tools provided you:

  • Give explicit notice of which tool was used and how
  • Rigorously test and alter AI-generated code
  • Understand all submitted code and can explain it
  • Write all comments and prose in your own words

Attendance

Two unexcused absences are permitted without grade penalty. Complete readings before each class and contribute to discussion. No laptops in class except for accessibility accommodations.

Accessibility & Wellness

Register with Student Accessibility Services (SAS) and request an accommodation email. Religious observance conflicts should be raised by end of week 2. Counseling and wellness resources are available through the Student Wellness Center.

Course Materials

No required textbooks. All readings are available via Canvas, GitHub, or direct hyperlinks. Python Jupyter notebooks are provided for every workshop session.

Office Hours

Wednesday 9–10 AM in Silsby 124 or via Zoom. Additional peer tutoring sessions will be announced on Slack.
